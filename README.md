# SMA Group 2 – Project 1

Data analysis project containing a Jupyter notebook that demonstrates a full EDA and statistical analysis workflow.

## Contents

| File | Description |
|---|---|
| `analysis.ipynb` | Main Jupyter notebook: data loading, EDA, visualisation, statistical testing |

## Getting Started

1. **Install dependencies**

   ```bash
   pip install numpy pandas matplotlib seaborn scipy
   ```

2. **Launch the notebook**

   ```bash
   jupyter notebook analysis.ipynb
   ```

3. Follow the cells from top to bottom. Replace the sample Iris dataset with your own data as needed.
